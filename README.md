# Heart Disease Prediction using Machine Learning

## Overview

This is my first end-to-end machine learning project where I built and compared multiple classification models to predict the likelihood of heart disease using patient health data. The project covers the complete machine learning workflow, including data preprocessing, exploratory data analysis, model training, evaluation, and prediction.

## Objective

The objective of this project is to develop a machine learning model that can predict whether a person is likely to have heart disease based on various medical attributes. Along the way, I explored the data, trained different models, and compared their performance to identify the most effective approach.

## Dataset

The dataset contains various patient health attributes, including:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise-Induced Angina
* ST Depression
* Slope of ST Segment
* Number of Major Vessels
* Thalassemia

**Target Variable**

* **0** – No Heart Disease
* **1** – Heart Disease
